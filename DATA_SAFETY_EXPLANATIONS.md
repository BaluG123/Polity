## üîí **SECURITY PRACTICES EXPLANATIONS**

### **Data Encryption in Transit**
```
All data transmitted between the app and our servers is encrypted using industry-standard HTTPS/TLS protocols. This ensures that user data cannot be intercepted or accessed by unauthorized parties during transmission.
```

### **Data Encryption at Rest**
```
User data is stored using Firebase Firestore, which provides automatic encryption at rest. This means all stored data is encrypted and protected even when not being actively accessed.
```

### **User Data Deletion**
```
Users can delete their accounts and all associated data through the app settings. Additionally, users can contact our support team to request data deletion. Upon account deletion, all personal data including quiz scores, progress, and bookmarks are permanently removed from our systems.
```
